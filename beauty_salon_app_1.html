Firebase 設定

📋 設定手順（約5分）

  1. console.firebase.google.com をGmailでログイン
  2. 「プロジェクトを追加」→ 好きな名前を入力 → 作成
  3. 左メニュー「Firestore Database」→「データベースの作成」
    →「テストモードで開始」→ 地域は asia-northeast1（東京）
  4. 左上の歯車⚙️「プロジェクトの設定」→「アプリを追加」→ ウェブ </> を選択
  5. アプリ名（例：salon）を入力 →「アプリを登録」→ 表示された値を下にコピー

⚠️ Firestoreのセキュリティルール設定（必須）

Firebase Console → Firestore Database → 「ルール」タブ → 以下をコピーして「公開」をクリック：

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } }

🌐 このアプリを複数のスマホで使うには？

HTMLファイルをウェブ上に公開する必要があります。Netlify Drop（無料）がおすすめです：
① このHTMLファイルを Netlify Drop にドラッグ&ドロップ
② 生成されたURLを2人のスマホで開く

設定はこのデバイスのブラウザに保存されます。
山本くるみさんのスマホでも同じ手順で設定してください。
